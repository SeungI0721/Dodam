# 문서 목록

이 폴더는 도담도담 Android 앱의 2026년 복구와 기능 보강 작업을 커밋용으로 정리한 문서 모음이다.

## 읽는 순서

1. `../README.md`
   - 프로젝트 개요, 실행 방법, 2021년 코드와 2026년 코드의 차이를 확인한다.

2. `phase0-audit.md`
   - 초기 감사 결과와 제거가 필요했던 위험 항목을 확인한다.

3. `phase1-build-recovery.md`
   - Gradle, SDK, Firebase 빌드 복구 내용을 확인한다.

4. `phase2-to-phase7-implementation.md`
   - 구조 개선, 장치 추상화, 대시보드, 설정, 알림, 기록, 인증 구현 내용을 확인한다.

5. `software-implementation-without-chatbot.md`
   - 챗봇을 제외하고 실제로 구현한 SW 기능만 확인한다.

6. `final-verification-report.md`
   - 최종 빌드, 테스트, lint, 미검증 항목을 확인한다.

7. `renamed-items.md`
   - 코드 구조 개선 과정에서 변경한 이름과 유지한 이름을 확인한다.

## 문서 작성 기준

- 실제 실행 가능한 상태와 미검증 상태를 분리해서 기록한다.
- 과거 문제는 감사 기록으로 남기되, 현재 제거된 항목은 제거 완료로 표시한다.
- 챗봇은 이번 구현 범위에서 제외했음을 명확히 표시한다.
- API key 원문이나 비밀정보는 문서에 적지 않는다.
